# PokeScout

Inventory tracker for Pokémon cards and sealed product.

## Projects
- **PokeScout.Api** – ASP.NET Core Web API (OpenAPI + Scalar)
- **PokeScout.Core** – Domain models / business rules
- **PokeScout.Infrastructure** – Data access, persistence
- **PokeScout.Web** – Frontend (TBD)
- **PokeScout.Tests** – Unit/integration tests

## Run the API
From Visual Studio: run the `https` profile.

Scalar UI:
- `/scalar`

OpenAPI JSON:
- `/openapi/v1.json`
